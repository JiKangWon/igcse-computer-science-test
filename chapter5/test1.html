IGCSE Computer Science Practice Test

(The Internet and Cyber Security)

Maximum Marks: 40
Recommended Time: 45 Minutes

SECTION A: Online Retail and Data Management

Scenario: EasyBuy Ltd. is a large international online retail company. They rely heavily on the Internet to connect with customers and process transactions.

Question 1: Accessing the Website and User Experience

a) Explain the fundamental difference between the Internet and the World Wide Web (WWW). [2]

b) A customer types the following address:
https://www.easybuy.co.uk/electronics/product_details.html

i. Identify the protocol being used and explain its purpose. [2]

ii. State the domain type and the file name. [2]

c) When a customer visits EasyBuy, the website sends a small file to their browser to remember preferences and shopping cart items.

i. Name this small file. [1]

ii. Describe the difference between a session cookie and a persistent cookie in this context. [2]

SECTION B: Digital Finance and Blockchain Security

Scenario: SecurePay is a new financial technology company that offers customers secure, decentralised payment options using cryptocurrency.

Question 2: Digital Currency and Blockchain

a) Explain how cryptocurrency overcomes the issues associated with the centralisation of traditional digital currency. [2]

b) SecurePay uses blockchaining to track transactions.

i. Describe the structure of a single block in a blockchain. [3]

ii. Explain how the use of hash values prevents a hacker from tampering with data. [3]

c) Explain the purpose of proof-of-work in slowing down the process of creating new blocks. [2]

SECTION C: Cyber Security Threats and Defences

Scenario: Global Data Solutions manages sensitive client data and faces constant cyber security threats.

Question 3: Security Threats and Mitigation

a) Identify and explain two distinct differences between a worm and a virus. [2]

b) An employee sees a pop-up warning of infection, advising them to download "SafeGuard Antivirus Trial".

i. Name the type of cyber security threat that relies on deceiving the user based on fear. [1]

ii. Name and describe the specific type of malware often disguised as legitimate software. [2]

c) Employees must use strong passwords.

i. State three specific criteria that define a strong password. [3]

ii. A manager uses Manager123. Explain why this password is vulnerable to a brute force attack. [1]

d) The company implemented a system to filter network traffic.

i. Name this security measure. [1]

ii. Describe two specific functions this measure performs. [2]
