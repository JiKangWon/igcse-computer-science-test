IGCSE Computer Science Practice Test 2

Cyber Security and Network Deep Dive

Maximum Marks: 40
Recommended Time: 45 Minutes

SECTION A: Network Management and Web Access

Scenario: FutureLearn Academy relies on a robust internal network to deliver online content to thousands of students.

Question 1: Network Infrastructure and Web Retrieval

a) The Academy uses a system to filter incoming and outgoing web traffic...

i. Name the security measure that performs this filtering function. [1]

ii. Describe two specific functions this measure performs to protect the internal network (other than blocking undesirable IP addresses). [2]

b) ...When a student types the portal’s URL, the student’s browser needs to find the server’s IP address.
Describe the vital role of the Domain Name Server (DNS) in converting the URL into the required IP address and facilitating the retrieval of the web page. [3]

c) The IT Manager decides to implement a Proxy Server.
Explain how using a proxy server's cache feature can significantly improve the speed at which students access frequently visited parts of the online learning portal. [2]

Question 2: Data Interception and Security Protocols

FutureLearn Academy uses both wired and wireless connections, increasing the risk of data interception.

a) ...A malicious individual attempts to steal data by tapping into a wired communication link.

i. Name this technique where the criminal examines data packets. [1]

ii. Explain how wardriving (or Access Point Mapping) is carried out by a hacker. [2]

b) When staff access sensitive data, they must use the HTTPS protocol.
Describe how the SSL certificate is used by the user's browser to authenticate the web server during the process of establishing a secure connection. [2]

SECTION B: Cyber Security Threats

Scenario: QuickTrans Logistics is a company whose employees handle sensitive shipping manifest data and financial transactions.

Question 3: Social Engineering and Password Protection

a) An employee, Tom, receives an urgent phone call... advising him to download specific software immediately.

i. Name the broad category of cyber threat that relies on manipulating people. [1]

ii. Identify the specific human emotion this threat relies upon. [1]

b) Another employee, Sarah, is locked out of her computer, and a message appears demanding payment...

i. Name the specific type of malware responsible for this attack. [1]

ii. Explain the most effective method for the company to avoid paying the ransom. [1]

c) ...The company's old password policy allowed passwords like Manager123.
Explain why even a complex password can still be vulnerable to a brute force attack if it is based on common sequences or dictionary words. [1]

Question 4: Phishing and Pharming

a) QuickTrans Logistics conducts training to distinguish between phishing and pharming.

i. Describe the main difference between phishing and pharming regarding the action required by the user. [2]

ii. State two clues that suggest an email is a potential phishing scam. [2]

b) Pharming often involves DNS cache poisoning.
Explain what DNS cache poisoning does to the legitimate IP address. [2]

SECTION C: High Security and Digital Identity

Scenario: VaultBank is a high-security financial institution that uses advanced authentication methods.

Question 5: Advanced Authentication

a) VaultBank uses two-step verification.
Explain how two-step verification works, using the three common factors of authentication in your description. [3]

b) To enter the physical data vaults, staff must use biometrics.

i. State two advantages of using fingerprint scans over traditional password or access card systems. [2]

ii. State one specific drawback of using retina scans for authentication, based on user experience. [1]

c) A staff member suggests using voice recognition.
Provide one benefit and one drawback of using voice recognition as a primary biometric security method. [2]
