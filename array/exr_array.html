Practice Problems

Exercise 1: Print First Element

Given the array colors = ["Red", "Green", "Blue"], write pseudo code to print the first element.

Exercise 2: Print Last Element

Given the array numbers = [10, 20, 30, 40, 50], write pseudo code to print the last element.

Exercise 3: Modify an Element

Given the array grades = [88, 75, 90], write pseudo code to change the second element (at index 1) to 80 and then print the entire array.

Exercise 4: Print All Elements

Given the array fruits = ["Apple", "Banana", "Cherry"], write pseudo code to print each element on a new line.

Exercise 5: Get User Input into Array

Write pseudo code that creates an empty array of size 3 and fills it with 3 numbers provided by the user.

Exercise 6: Calculate Sum

Given data = [5, 10, 15, 20], write pseudo code to calculate and print the sum of all its elements.

Exercise 7: Calculate Average

Given scores = [100, 80, 90, 70], write pseudo code to calculate and print the average score.

Exercise 8: Find the Smallest Number

Given values = [45, 22, 88, 12, 5], write pseudo code to find and print the smallest number in the array.

Exercise 9: Find the Largest Number

Given values = [45, 22, 88, 12, 5], write pseudo code to find and print the largest number in the array.

Exercise 10: Count Even Numbers

Given nums = [1, 2, 3, 4, 5, 6], write pseudo code to count how many numbers are even.

Exercise 11: Linear Search

Given names = ["Ana", "Tom", "Lee"], write pseudo code to check if the name "Tom" exists in the array. Print "Found" or "Not Found".

Exercise 12: Copy an Array

Given arrayA = [1, 2, 3], write pseudo code to copy all its elements into a new array called arrayB.

Exercise 13: Print Positive Numbers

Given mix = [5, -10, 15, -2, 8], write pseudo code to print only the positive numbers.

Exercise 14: Print Array in Reverse

Given items = [10, 20, 30, 40], write pseudo code to print the elements in reverse order (40, 30, 20, 10).

Exercise 15: Find Index of Smallest

Given data = [15, 8, 22, 10], write pseudo code to find the index of the smallest number (which is 8 at index 1).

Exercise 16: Check for Palindrome

An array is a palindrome if it reads the same forwards and backwards. Given arr = [1, 2, 3, 2, 1], check if it's a palindrome.

Exercise 17: Reverse an Array (In-Place)

Given arr = [1, 2, 3, 4, 5], write pseudo code to reverse the array *without* creating a new one. The final array should be [5, 4, 3, 2, 1].

Exercise 18: Check if Sorted

Given nums = [10, 20, 30, 45, 50], write pseudo code to check if the array is sorted in ascending (increasing) order.

Exercise 19: Shift Elements Left

Given arr = [1, 2, 3, 4], "shift" all elements one position to the left and move the first element to the end. The result should be [2, 3, 4, 1].

Exercise 20: Concatenate Two Arrays

Given listA = [1, 2] and listB = [3, 4], write pseudo code to create a new array listC that contains all elements from A, followed by all elements from B. The result should be [1, 2, 3, 4].
