Question 1: Digital Media Storage (Data Representation)

Scenario: "Artisan Media" specializes in creating high-resolution bitmap images and sound clips for video games.

a) An image has a resolution of 512 × 512 pixels and can represent 65,536 different colours.

i. Calculate the colour depth, in bits, used for this image. [1]

ii. Calculate the file size of this image in KiB. (Show working. $1 \text{ KiB} = 2^{10}$ bytes). [3]

b) The studio uses the hexadecimal colour code #FF9966.

i. Explain why hexadecimal is used instead of denary for colour codes. [1]

ii. Convert the hexadecimal value FF into its equivalent denary value. [1]

c) A positive denary number, 59, is loaded into an 8-bit register.

i. Convert denary 59 into an 8-bit binary number. [1]

ii. The binary number for 59 undergoes a logical shift two places to the left. Show the 8-bit result and state the new denary value. [2]

d) Sampling sound at a high rate results in large files.

i. State two factors (other than length) that increase a sound file's size. [2]

ii. Describe the difference between lossy and lossless compression, giving one example for each. [4]

Question 2: Inter-Office Communication (Data Transmission)

Scenario: "GlobalLink" corporation transfers large, sensitive financial reports between its headquarters (City A) and a satellite office (City B) 500 km away.

a) Large reports are broken into smaller units.

i. Name this small unit and identify its three main sections. [2]

ii. Explain the purpose of the sequence number in the header. [1]

iii. Describe how a Cyclic Redundancy Check (CRC) detects errors. [2]

b) GlobalLink uses full-duplex transmission.

i. Explain what full-duplex transmission means. [1]

ii. State the transmission type (serial/parallel) used by USB and explain why it is suitable for external devices compared to the alternative. [3]

c) Communication is encrypted.

i. Identify the encryption method that uses a public key and a private key. [1]

ii. Describe how this method ensures only City B can decrypt the document sent by City A. [3]

d) An ISBN-13 code is entered: 9780340983829.
Explain the purpose of the final digit (9) and identify this type of error-checking mechanism. [2]

Question 3: Online Retail and Security (Internet/Cyber Security)

Scenario: "BuySafe," an online retailer, handles thousands of transactions daily and must maintain a robust and secure online presence.

a) A customer types a URL into their browser.

i. Explain the role of the Domain Name Server (DNS). [2]

ii. The BuySafe website uses HTTPS. Explain the difference between HTTP and HTTPS and identify the security protocol used. [2]

b) BuySafe uses cookies.

i. Define the term cookie. [1]

ii. Describe the key difference between a session cookie and a persistent cookie. [1]

c) BuySafe is battling cyber threats.

i. Describe the process used in a Brute Force Attack. [1]

ii. A customer's screen is locked with a payment demand. Name this malware. [1]

iii. Explain how Two-step Verification improves security. [2]

d) BuySafe uses a firewall and a proxy server.
Describe two ways a firewall contributes to network security. [2]
