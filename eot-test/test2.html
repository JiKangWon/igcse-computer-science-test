Section A: Data Handling using One-Dimensional Arrays

Question 1: The Local Coffee Shop Inventory

The Local Coffee Shop needs a simple computer system to manage its 15 best-selling menu items and track daily sales volumes.

a) The manager requires two one-dimensional arrays to permanently store the name and price for the 15 items.
Declare two arrays in pseudocode suitable for storing 15 items, ensuring they can hold the appropriate data type. [2]

b) The system must allow staff to input the item name and price for all 15 inventory slots.
Write an algorithm, using iteration, that prompts the user to input the Name and Price for each of the 15 items and stores this data in the arrays declared in part (a). [4]

c) On a busy Saturday, the shop sells a variable quantity of each item. The quantity sold for each item is stored in a separate one-dimensional array called QuantitySold.
Write a Function called CalculateRevenue that takes the Price array and the QuantitySold array as parameters. The function should calculate and return the total revenue generated from all 15 items. [5]

d) The staff needs a quick way to check the current price of a specific item.
Write a Procedure called SearchPrice that takes the Name array and the Price array as parameters. The procedure should:

  • Prompt the user to enter the name of the item they wish to find.
  • Use a Linear Search algorithm to check if the item is present in the Name array.
  • If the item is found, output the item's name and its corresponding price.
  • If the item is not found, output an appropriate error message. [6]

Section B: Handling Complex Data Structures (2D Arrays)

Question 2: High Achievers Academy Grade Management

High Achievers Academy tracks the end-of-year scores for 10 students across 5 core subjects. All scores are whole numbers. The results are stored in a 2D array Scores (10 rows, 5 columns) and a 1D array StudentNames.

a) Declare the necessary arrays in pseudocode to store the student names (10 students) and the examination scores (10 students, 5 subjects). [2]

b) Describe, using appropriate terminology, how nested iteration would be used to input scores for all 10 students across all 5 subjects into the Scores array. (You do not need to write the full pseudocode, only explain the process.) [3]

c) The principal needs to know the average score achieved in the 5th subject (IT).
Write a Function called SubjectAverage that takes the Scores array and the subject index (e.g., 5) as parameters. The function must calculate and return the average score for that specific subject. (Assume 1-based indexing). [4]

d) The academic committee wishes to identify the single highest score achieved by any student across all 5 subjects, along with the student index and subject index where that score was recorded.
Write an algorithm in pseudocode to find and output the maximum score, the row index (StudentNumber), and the column index (SubjectNumber) corresponding to that score. (Assume 1-based indexing). [6]

e) The Academy wants to rank students based on their cumulative average. Assume total scores are already stored in a 1D array TotalScores.
Explain how a Bubble Sort algorithm could be applied to rearrange the StudentNames array so that the names are listed in descending order of their total scores. [3]
