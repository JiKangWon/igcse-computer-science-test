Computer Science IGCSE Practice Test

Chapter Review (Data Transmission)

Instructions: Answer all questions. The number of marks is given in brackets [ ] at the end of each question.

Scenario 1: Global E-commerce Network

Scenario: WebShop is a large e-commerce company. When a customer places an order, the data is sent from the customer’s device, across the Internet, to WebShop’s servers... The data travels through an interconnected network involving many routers.

Question 1

a) The customer order data is broken down into smaller units before transmission.
Identify the name given to these small units and describe the three primary sections that make up each unit. [4]

b) WebShop uses packet switching to transmit these units.
Explain how packet switching ensures that the data can reach the server even if a particular transmission path becomes busy or fails. [3]

c) Due to the sensitive nature of the payment information, WebShop uses encryption.
Explain the primary purpose of encryption in this context. [2]

d) WebShop implements asymmetric encryption...
Describe how two different keys (Public and Private) are used by the sender and the receiver to ensure that only the intended server can read the message. [4]

Scenario 2: Automated Warehouse System

Scenario: AutoLogistics operates a fully automated warehouse. Inventory data is constantly sent wirelessly. Ensuring this data is accurate is critical.

Question 2

a) To check for errors during transmission, AutoLogistics uses a Checksum method.
Describe the steps involved in using a checksum to verify that a block of data has arrived without transmission errors. [3]

b) AutoLogistics sometimes uses Automatic Repeat Requests (ARQs).
Explain how ARQ ensures reliable data transfer by describing the roles of acknowledgements and timeout. [3]

c) When new product codes are entered manually, a check digit is used.

i) Define the term check digit and state one type of error it detects. [2]

ii) Distinguish between a checksum and a check digit based on their purpose. [3]

Scenario 3: University Computer Lab

Scenario: A computer science department is designing the internal architecture and peripheral connections for its new teaching lab.

Question 3

a) Data is transmitted between the CPU and RAM. This must be extremely fast over short distances.
Identify the most appropriate type of data transmission (Serial or Parallel) and justify your choice. [2]

b) The university uses Universal Serial Bus (USB) connections.

i) State the transmission method (Serial or Parallel) used by USB. [1]

ii) Explain two benefits of using the USB standard. [4]

c) The network card allows students to download videos while simultaneously uploading commands.
Identify this data transmission mode and describe it. [2]

d) Older equipment uses half-duplex transmission.
Provide a clear example of a half-duplex scenario (outside the lab) and explain why it is appropriate. [2]
