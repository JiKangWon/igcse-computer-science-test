Computer Science IGCSE Practice Test 2

Data Security and Transmission

Instructions: Answer all questions. The number of marks is given in brackets [ ] at the end of each question.

Scenario 1: Secure Patient Record System

Scenario: MedData Corp manages sensitive patient records, which are frequently transmitted between local clinics and a central secure database. Data integrity and security are paramount.

Question 1

a) Explain the fundamental purpose of encryption when sensitive data, such as patient records, is transmitted. [2]

b) MedData Corp uses symmetric encryption for internal communications.

i) Describe how symmetric encryption works, focusing on the roles of the encryption key and the decryption key. [2]

ii) State and explain the major security drawback associated with symmetric encryption. [2]

c) The system uses a parity check method for error detection.

i) Define the term parity bit and explain its role. [2]

ii) Assume the system uses Odd Parity. The byte 0 1 1 0 1 0 0 1 is received. Show your working to determine if a transmission error occurred. [2]

Scenario 2: Automated Vehicle Registration

Scenario: The National Vehicle Registry (NVR) processes vehicle identification numbers (VINs). Staff manually enter these long codes.

Question 2

a) To catch human input errors, the NVR uses a check digit system.

i) Explain how a check digit is used to detect an error like a transposition error. [2]

ii) Distinguish between a checksum and a check digit based on their purpose and when they are applied. [3]

b) The local NVR branch uses a two-way radio system (like a walkie-talkie).
Identify the transmission mode used by this system and describe its characteristic operation. [2]

c) The main NVR server connects to broadband, allowing it to download updates while simultaneously uploading reports.
Identify the transmission mode for this connection and explain why it is necessary. [2]

Scenario 3: Large File Transfer Protocol

Scenario: When a user downloads a large software update from the NVR website, the file is first broken down into data packets before transmission.

Question 3

a) Identify the three main sections of a data packet and list one piece of essential information contained within the packet header. [3]

b) Explain two benefits of splitting a large file into small packets (around 64 KiB) in a busy network. [4]

c) The Cyclic Redundancy Check (CRC) is performed on arrival.
Describe how the CRC (in the trailer) is used to verify the integrity of the payload. [3]

d) The NVR network uses an Automatic Repeat Request (ARQ) protocol.
Describe the role of acknowledgement and timeout when a packet is corrupted or lost. [3]
