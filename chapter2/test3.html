Computer Science IGCSE Practice Test 3

Network Fundamentals and Security

Instructions: Answer all questions. The number of marks is given in brackets [ ] at the end of each question.

Scenario 1: International Financial Transaction

Scenario: GlobalBank processes millions of secure financial transfers daily... All data, including customer account numbers and transfer amounts, is sent across the public internet.

Question 1

a) ...data is broken down into data packets before transmission.

i) Explain two benefits for GlobalBank of splitting the large financial instruction into small packets. [2]

ii) Identify the three main sections of a data packet. [3]

iii) State two pieces of essential information contained within the packet header. [2]

b) GlobalBank uses packet switching...
Describe the primary role of a router in deciding the path of a data packet. [2]

c) ...GlobalBank employs encryption.

i) State the purpose of encryption when data is transmitted over a public network. [2]

ii) Explain the fundamental difference between symmetric and asymmetric encryption. [3]

Scenario 2: Remote Weather Station Data

Scenario: Automated Weather Stations send hourly blocks of data to a central processing hub (CPH). Data integrity upon arrival is crucial.

Question 2

a) The Weather Stations use Checksum after transmitting each block.
Explain how a checksum is used to verify that a block of data has arrived without corruption. [3]

b) ...two bits in a single byte may flip during transmission.
Explain why a simple parity check might fail to detect this error. [2]

c) ...the CPH sometimes requests an echo check.

i) Describe how the echo check process works. [2]

ii) Explain why an echo check is considered not very reliable. [1]

d) ...a check digit is used.
Explain the primary type of error check digits detect, and contrast this with checksums. [3]
