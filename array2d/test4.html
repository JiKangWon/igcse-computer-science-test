Problem Solving with 2D Arrays

Pseudocode Challenge: The Quarterly Sales Report

Scenario: A sales manager tracks the monthly sales for their team of 4 salespeople over one quarter (3 months: Jan, Feb, Mar). The data is stored in two parallel arrays:
  • SalesTeam: A 1D array of strings storing the names. (Size 4)
  • SalesData: A 2D array of integers storing the sales figures. (Size 4x3)

The row index links the two arrays (e.g., SalesTeam[1] is the name for the sales data in SalesData[1]). The column index represents the month (0=Jan, 1=Feb, 2=Mar).

The Task

Write a pseudocode algorithm that processes the sales data and meets the following requirements:

  1. The algorithm must assume that the SalesTeam (1D array, size 4) and SalesData (2D array, size 4 rows, 3 columns) are already populated with data.
  2. Declare a new 1D array named SalesTotals capable of holding 4 integer values, one for each salesperson's total.
  3. Declare two variables: HighestTotal (integer, initialized to 0) and TopPerformer (string, initialized to "").
  4. The program must use nested loops to traverse the SalesData array.
    • The outer loop must iterate through the rows (salespeople, index 0 to 3).
    • The inner loop must iterate through the columns (months, index 0 to 2).
  5. Inside the inner loop, the algorithm must add the sales figure from SalesData[row][col] to a running total for that specific row.
  6. After the inner loop finishes (but still inside the outer loop), the algorithm must:
    • Store the final running total for that row into the corresponding index of the SalesTotals array (e.g., SalesTotals[row]).
    • Immediately print the salesperson's name and their total sales for the quarter.
    • Check if this salesperson's total (from SalesTotals[row]) is greater than HighestTotal. If it is, update HighestTotal with the new value and TopPerformer with the salesperson's name (from SalesTeam[row]).
  7. After all loops are complete, the program must display the overall top performer and their total sales under a separate heading.

Example Output Flow

(Assuming SalesTeam = ["Anna", "Ben", "Chris", "Dana"])

*** QUARTERLY SALES TOTALS ***
Anna: $1550
Ben: $1200
Chris: $1850
Dana: $1600

*** TOP PERFORMER ***
Chris: $1850
