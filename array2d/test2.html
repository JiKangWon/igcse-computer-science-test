Coding Problems

Topic: 1D and 2D Arrays

30 Coding Challenges

Write pseudo code or a program to solve each of the following problems.

Problem 1: Find the Sum

Given an array of integers, write a program to calculate and print the sum of all its elements.

Example input: [10, 20, 30, 40]

Problem 2: Find the Largest Element

Given an array of integers, find and print the largest element in the array.

Example input: [85, 92, 78, 95, 88]

Problem 3: Find the Smallest Element

Given an array of integers, find and print the smallest element in the array.

Example input: [30, 22, 25, 29, 21]

Problem 4: Calculate the Average

Given an array of numbers, calculate and print their average.

Example input: [10, 5, 15, 20]

Problem 5: Count Occurrences

Given an array of strings and a target string, count and print how many times the target string appears in the array.

Example input: array = ['A', 'B', 'A', 'C', 'A'], target = 'A'

Problem 6: Print in Reverse

Given an array, print all its elements in reverse order.

Example input: [1, 2, 3, 4, 5]

Problem 7: Count Even Numbers

Given an array of integers, count and print how many of them are even numbers.

Example input: [1, 2, 3, 4, 5, 6]

Problem 8: Count Positive Numbers

Given an array of integers, count and print how many of them are greater than 0.

Example input: [-1, 1, 5, -10, 3]

Problem 9: Find Element (Linear Search)

Given an array and a target value, print "Found" if the target exists in the array, and "Not Found" otherwise.

Example input: array = [1, 3, 5, 7], target = 5

Problem 10: Swap First and Last

Given an array, swap its first and last elements. Print the modified array.

Example input: [10, 20, 30, 40]

Problem 11: Create a Copy

Given an array arrayA, create a new array arrayB that is a perfect copy of arrayA.

Example input: [1, 2, 3]

Problem 12: Reverse an Array (In-Place)

Given an array, reverse its elements *without* creating a new array (this is called "in-place").

Example input: [1, 2, 3, 4, 5]
Expected output: [5, 4, 3, 2, 1]

Problem 13: Check for Duplicates

Given an array of integers, print "True" if the array contains any duplicate values, and "False" otherwise.

Example input 1: [1, 2, 3, 1] (Should print True)
Example input 2: [1, 2, 3, 4] (Should print False)

Problem 14: Merge Two Arrays

Given two arrays, create a new array that contains all elements from the first array, followed by all elements from the second array.

Example input: arrayA = [1, 2], arrayB = [3, 4]

Problem 15: Find Second Largest

Given an array of integers, find and print the second largest number. (Assume all numbers are unique and the array has at least 2 elements).

Example input: [85, 92, 78, 95, 88]

Problem 16: Print a 2D Array

Given a 2D array (matrix), write a program to print all its elements, one row at a time.

Example input: [ [1, 2, 3], [4, 5, 6] ]

Problem 17: Sum of All Elements (2D)

Given a 2D array of integers, calculate and print the sum of *all* its elements.

Example input: [ [10, 20], [30, 40] ]

Problem 18: Find Largest Element (2D)

Given a 2D array of integers, find and print the largest element in the entire matrix.

Example input: [ [1, 5, 2], [8, 3, 6] ]

Problem 19: Sum of a Specific Row

Given a 2D array and a row index R, calculate and print the sum of all elements in row R.

Example input: matrix = [ [1, 2], [3, 4], [5, 6] ], R = 1

Problem 20: Sum of a Specific Column

Given a 2D array and a column index C, calculate and print the sum of all elements in column C.

Example input: matrix = [ [1, 2], [3, 4], [5, 6] ], C = 0

Problem 21: Sum of Each Row

Given a 2D array, calculate and print the sum of *each* row, one by one.

Example input: [ [80, 85, 90], [92, 88, 95] ]
Expected output:
Row 0 Sum: 255
Row 1 Sum: 275

Problem 22: Sum of Each Column

Given a 2D array, calculate and print the sum of *each* column.

Example input: [ [1, 2, 3], [4, 5, 6] ]
Expected output:
Col 0 Sum: 5
Col 1 Sum: 7
Col 2 Sum: 9

Problem 23: Sum of the Main Diagonal

Given a square 2D array (e.g., 3x3), calculate and print the sum of its main diagonal (from top-left to bottom-right).

Example input: [ [1, 2, 3], [4, 5, 6], [7, 8, 9] ]
(Elements to sum are 1, 5, and 9)

Problem 24: Sum of the Anti-Diagonal

Given a square 2D array (e.g., 3x3), calculate and print the sum of its anti-diagonal (from top-right to bottom-left).

Example input: [ [1, 2, 3], [4, 5, 6], [7, 8, 9] ]
(Elements to sum are 3, 5, and 7)

Problem 25: Count Target Value (2D)

Given a 2D array and a target value, count and print how many times the target appears in the matrix.

Example input: matrix = [ [1, 5, 1], [1, 2, 5] ], target = 1

Problem 26: Find Row with Largest Sum

Given a 2D array, find and print the index of the row that has the largest sum.

Example input: [ [1, 2, 3], [10, 10, 10], [4, 5, 6] ]
(Row 0 sum=6, Row 1 sum=30, Row 2 sum=15. Output should be 1)

Problem 27: Create a 2D Array (User Input)

Write a program that asks the user for a number of rows R and columns C, then populates a 2D array by asking the user for each element's value. Finally, print the resulting 2D array.

Problem 28: Flatten a 2D Array

Given a 2D array, create a new 1D array that contains all elements of the 2D array, in row-by-row order.

Example input: [ [1, 2], [3, 4] ]
Expected output: [1, 2, 3, 4]

Problem 29: Transpose a Matrix

Given a 2D array, create a new 2D array that is its "transpose" (rows become columns, and columns become rows).

Example input: [ [1, 2, 3], [4, 5, 6] ]
Expected output: [ [1, 4], [2, 5], [3, 6] ]

Problem 30: Check if Square Matrix is Symmetric

A square matrix is symmetric if it is equal to its transpose (i.e., matrix[i][j] == matrix[j][i] for all i, j). Write a program to check if a given square matrix is symmetric.

Example input 1: [ [1, 2], [2, 1] ] (Symmetric)
Example input 2: [ [1, 2], [3, 4] ] (Not Symmetric)
