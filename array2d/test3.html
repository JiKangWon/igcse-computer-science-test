Pseudocode Challenge: The Final Exam Audit

Scenario: A professor needs to audit the final exam results for a class of 50 students. They have analyzed the class performance and stored the data in two parallel 1D arrays:
  • StudentNames (stores the name of each student, e.g., "Anh Nguyen")
  • ExamScores (stores the final exam score for each student, e.g., 85)

The professor needs a program that processes this data to identify two groups, as well as calculate class statistics.

The Task

Write a pseudocode algorithm that processes the exam data and meets the following requirements:

  1. The algorithm must assume that the two parallel arrays, StudentNames and ExamScores, both containing 50 elements, are already populated with data.
  2. Declare and initialize a third 1D array named FailingStudents capable of holding 50 student names (strings). Also, declare and initialize an integer counter named FailCount to 0.
  3. Declare and initialize integer variables ClassTotal (for summing all scores) and HighestScore (for tracking the maximum score) to 0.
  4. The program must search through the ExamScores array (from index 0 to 49).
  5. For every student, the algorithm must perform the following actions:
    • Add the student's score to the ClassTotal.
    • Check if the student's score is greater than the current HighestScore and update it if necessary.
    • Top Student Output: If the ExamScores[i] is greater than or equal to 90, the algorithm must immediately display the student's name and their score under the heading "*** TOP STUDENTS (>= 90) ***".
    • Failing Student Tracking: If the ExamScores[i] is less than 50, the algorithm must store the student's name (from StudentNames[i]) into the next available position in the FailingStudents array and increment the FailCount counter.
  6. After the search loop is complete, the program must use a separate section to display the final statistics:
    • Calculate and display the ClassAverage (ClassTotal / 50).
    • Display the HighestScore.
    • Use a separate loop to display the contents of the FailingStudents array under the heading "*** FAILING STUDENTS (< 50) ***".
    • If the FailCount is 0, the program must display the message: "No students failed the exam." instead of the list.
